# 🚦 Suspicious Human Activity and Traffic Violation Detection System

An AI-powered surveillance system that detects suspicious human activities and traffic violations in real time using deep learning and computer vision techniques.

---

## 📖 Overview

This project implements a real-time intelligent monitoring system using YOLO-based object detection models. The system analyzes live or recorded video streams to identify:

- Suspicious human activities
- Traffic rule violations
- Vehicle and pedestrian detection

It aims to enhance public safety through automated surveillance and smart monitoring solutions.

---

## 🎯 Features

- Real-time object detection using YOLOv5
- Detection of traffic violations:
  - Helmet violations
  - Signal jumping
  - Unauthorized lane usage
- Suspicious behavior detection
- Supports both video files and live camera feed
- Bounding box visualization with confidence scores

---

## 🧠 Technologies Used

- Python
- YOLOv5 / Ultralytics
- PyTorch
- OpenCV
- NumPy
- Matplotlib
